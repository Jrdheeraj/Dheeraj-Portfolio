## About

This is a personal portfolio website built with React 18 and TypeScript, featuring smooth animations and a clean, modern design. It showcases five major projects including TruthLens AI (multimodal fact verification), GeoAI (urban growth monitoring), Solar Scan AI, FactoryFix AI, and OxyWalls (smart green wall air purifier). The portfolio is fully responsive and optimized for all devices.

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui
- **Animation**: Framer Motion, Lenis Scroll
- **Build Tool**: Vite
- **Testing**: Vitest

## Features

- Smooth scroll animations and transitions
- Interactive project showcase with detailed descriptions
- Achievements & certifications section
- Fully responsive design
- Optimized performance

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or bun

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/MAHESHPPAI/portfolio-website.git
   cd Portfolio-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   bun install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
